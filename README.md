# My all-in-one instructions on how to install Fedora Linux the easy way.

## You can use this order:
1. **`dualboot-install.md`** (install Fedora alongside your Windows OS, with no risk)
2. Post install script:
```bash
  sh -c "$(curl -fsSL https://raw.githubusercontent.com/z-Eduard005/fedora-install/main/post-install.sh)"
```
3. **`programs/*.md`** (other program settings recommendations)

### Fedora can be easily installed without any help, but I think this instruction and script will be helpful for Linux beginners and can save some time figuring out things you don't even expect in a new OS.

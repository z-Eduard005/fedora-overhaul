# Installation near windows os

## Windows part
1. Fully update system with system updates and microsoft store updates 
2. Disable fast startup: "Win + R" -> `powercfg.cpl`
3. Disable indexing on every disk (optional)
4. Disable bitlocker on every disk
5. Make free space partion with default windows utilitty (at least 256gb)
6. Reboot
7. Install fedora media writer utilitty: https://github.com/FedoraQt/MediaWriter/releases/latest and write last fedora version to thumbstick

## Bios/uefi part
1. Disable secure boot
2. Reboot into windows to check its working and go back to bios/uefi
3. Choose fedora thumbstick and boot into it

## Fedora part
1. Test all the things: camera, micro, sound, headphones, charging, monitor refresh rate, keyboard lightning, wifi, bluetooth, etc.
2. Open fedora installer app:
  2.1. 
3. Reboot into windows to check its working and go back to fedora
4. Use **`post-install.md`** instruction next

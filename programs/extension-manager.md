# Extension Manager

*Adds a bunch of community-created GNOME extensions*

## Must-have programs
1. **AppIndicator and KStatusNotifierItem Support** (adds system tray icons)
2. **Quick Lang Switch** (removes language switch popup to stay windows focused when changing language)
3. **Just Perfection** (more like another system settings app, but here you can disable almost every aspect of the system you don't like)
4. **Blur my Shell** (adds blur to your system)
5. **Dash to Dock** (adds a fully customizable dock like on Mac)
6. **Dash to Panel** (adds a fully customizable panel like on Windows)
7. **Gtk4 Desktop Icons NG (DING)** (adds desktop icons)


## Optional
1. **Open Bar** (customize your top bar)
2. **Vicinae** (for managing GNOME clipboard) **`programs/vicinae.md`**
3. **Color Picker** (pick a color on your screen)
4. **Vitals** (adds a panel to the top bar with system information like CPU usage)
5. **Rounded Window Corners Reborn** (change corners, border and shadow for all windows in the system)
6. **Hide Top Bar** (hides the top bar if it bothers you)

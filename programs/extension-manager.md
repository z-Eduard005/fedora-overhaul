# Extension Manager

*Adds a bunch of community-created GNOME extensions*

## Must-have programs
1. **AppIndicator and KStatusNotifierItem Support** (adds system tray icons)
2. **Open Bar** (customize your top bar)
3. **Quick Lang Switch** (removes language switch popup to stay windows focused when changing language)
4. **Just Perfection** (more like another system settings app, but here you can disable almost every aspect of the system you don't like)
5. **Blur my Shell** (adds blur to your system)
6. **Dash to Dock** (adds a fully customizable dock like on Mac, or you can search how to turn it into a Windows panel)

## Optional
1. **Vicinae** (for managing GNOME clipboard) **`programs/vicinae.md`**
2. **Color Picker** (pick a color on your screen)
3. **Vitals** (adds a panel to the top bar with system information like CPU usage)
4. **Rounded Window Corners Reborn** (change corners, border and shadow for all windows in the system)
5. Hide Top Bar (hides the top bar if it bothers you)

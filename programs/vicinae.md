# Vicinae

*The best app launcher for linux. Fully competible with raycast ecosystem*

## Install
1. `curl -fsSL https://vicinae.com/install.sh | bash && systemctl --user enable vicinae --now`
2. GNOME tweaks -> Startup Applications -> + -> Vicinae
3. Settings -> Keyboard -> View and Customize Shortcuts -> Custom Shortcuts -> + -> command: `vicinae toogle`; shortcut: <alt+space>
  3.1. Settings -> Keyboard -> View and Customize Shortcuts -> Windows -> Active the window menu -> <shift+ctrl+alt+space>
4. Settings -> Keyboard -> View and Customize Shortcuts -> Custom Shortcuts -> + -> command: `vicinae deeplink vicinae://launch/clipboard/history`; shortcut: <alt+v>

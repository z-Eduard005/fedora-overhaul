# Kitty

*The best terminal app*

## Settings I use
1. `nano ~/.config/kitty/kitty.conf`:
    ```
    font_size 18.0
    hide_window_decorations yes
    background_opacity 0.95
    window_padding_width 10
    enable_audio_bell yes
    ```
2. You can move/resize windows with the pressed Super (Windows) key and the left/right mouse button  
3. Terminal utilities:  
  3.1. **zsh**: `sudo dnf in zsh && chsh -s $(which zsh)`  
  3.2. **oh-my-zsh**: `sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"`
4. It's also possible to set up themes and change your own colors

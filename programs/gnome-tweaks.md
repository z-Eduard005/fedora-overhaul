# GNOME tweaks

*Basically an app for expanding system-wide settings*

## Suggestions
1. Keyboard → Additional Layout Options → Switching to another layout → "Caps Lock"
*Really useful, because the Caps Lock key is not important at all*
2. You can easily add Startup Applications here as well
3. It's also possible to add minimize and maximize buttons; change fonts, cursor, theme, and icons; and adjust other system behavior
